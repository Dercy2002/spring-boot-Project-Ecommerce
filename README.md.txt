# 🛍️ Spring Boot E-commerce Project

Bienvenue dans le projet **E-commerce** développé avec **Spring Boot**, **Spring Security (JWT)**, et une base de données **MySQL**.

## 🔧 Technologies utilisées

- Java 17
- Spring Boot 3.x
- Spring Security + JWT
- MySQL
- JPA (Hibernate)
- Maven
- Bootstrap (pour le front-end)
- Postman / Swagger pour tester l'API

## 📦 Fonctionnalités

- 🔐 Authentification et enregistrement avec rôles (`USER` / `ADMIN`)
- 🛒 Gestion des produits
- 🛍️ Panier et commandes
- 📑 API REST sécurisée
- 🔄 Redirection selon le rôle après connexion
